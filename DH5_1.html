CSS Box Model – Visual Practice

Each box below shows how content, padding, border, and margin affect layout. Open DevTools (Right-click → Inspect) to see the box model diagram.

BOX 1 → Padding: 20px | Border: 2px solid | Margin: 20px
Content Area
This text is inside Box 1.
Padding creates space inside the border.
BOX 2 → Padding: 40px | Border: 4px dashed | Margin: 30px
Content Area
This box has more padding,
so the content is pushed further inside.
BOX 3 → Padding: 10px 40px | Border: 3px dotted | Margin: 15px 50px
Content Area
Left and right padding is larger
than top and bottom padding.

Key Takeaways

  • Padding increases space inside the box
  • Border surrounds content + padding
  • Margin controls spacing between elements
  • Total size = content + padding + border (margin is outside)
